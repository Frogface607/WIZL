# WIZL Monster Sticker Prompts for Freepik Spaces

Each strain gets a unique monster character. Style: cute but cool, collectible vinyl toy aesthetic.
Colors from WIZL palette. Dark background. "wizl.space" small text at bottom.

## Template
```
Cute monster character for "{STRAIN_NAME}" cannabis strain. The monster's personality reflects the strain: {VIBE_DESCRIPTION}. Small round creature, big expressive eyes, {COLOR_SCHEME} color scheme. Collectible vinyl toy style, dark background (#0a0a0e). Tiny text "wizl.space" at the bottom center in muted gray. No cannabis leaves. Sticker-ready design with slight drop shadow.
```

## How to use in Freepik Spaces
1. Copy one prompt at a time
2. Set aspect ratio to 1:1 (square for stickers)
3. Generate 2-3 variations
4. Pick the best one
5. Add "#001", "#002" numbering in post-production
6. Each monster = collectible NFT-style sticker

## Rarity System
- **Common** (green border): strains everyone knows
- **Uncommon** (blue border): popular but not mainstream
- **Rare** (purple border): connoisseur favorites
- **Epic** (gold border): legendary strains
- **Legendary** (rainbow border): the absolute classics
